RELACIÓN CON SERVIDOR**

Cada jugador al iniciar una partida nueva envia al servidor su pid que corresponderá al id del jugador.
Para comunicarnos con el servidor, enviaremos un mensajea que contenga:

- PID (id del jugador)
- Rol del jugador (guardián o explorador)
- Posición (estruct: posX, posY)
- Mensaje (si hubo colisiones especiales Ej: conseguír tesoro, explorador atrapado)

Nosotros (cliente) nos encargamos de:
Detectar bordes, obstáculos y colisiones especiales, para no sobrecargar el servidor con constantes solicitudes.

RELACIÓN CON DIRECTORIO** (LISTA)
Para la lista de mapas disponibles, necesitaremos comunicarnos con el directorio, el cual tiene la información:
SOLICITUD:
- Nuestro PID
- Tipo de opcion (listar, crear, etc)
- Texto (mensaje extra)

RESPUESTA RECIBIDA:
- PID (para saber cual nos corresponde)
- Código (ok o error)
- Número de catacumbas
- Datos de las catacumbas ej) Catacumba1|Dir1|Mailbox1|0|10;Catacumba2|Dir2|Mailbox2|5|20;...
